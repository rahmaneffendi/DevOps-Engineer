# Webserver

## Instalasi Nginx

disini saya memakai ansible seperti biasa

```
- hosts: gateway
  become: yes
  gather_facts: yes
  tasks:
        - name: 'install nginx'
          apt: 
            name:
             - nginx
            state: latest
```
